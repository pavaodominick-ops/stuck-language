Aula 1: Escopo Global

Todo o teu código precisa obrigatoriamente de ser escrito dentro do bloco inicial:

logical{\n  // código\n}

Aula 2: Variáveis e Comentários

Cria comentários usando // e declara variáveis dinâmicas usando o prefixo cot:

cot idade = 15\ncot ativo = true

Aula 3: Saída de Texto

Para exibir mensagens ou dados processados no console abaixo, usa a instrução:

aparec("Olá Mundo!")

Aula 4: Operações e Parênteses

Em STUCK, o segundo termo matemático ou condicional vem sempre envolvido por parênteses colados ao operador:

cot x = 10 +(5)\ncot check = x >(12)

Aula 5: Decisões (If / Else)

Desvia o fluxo do teu programa avaliando testes condicionais:

if(idade >=(18)){\n  aparec("Livre")\n}\nelse{\n  aparec("Barrado")\n}

Aula 6: Loops e Repetições

Usa repet(n) para repetição fixa, ou while e for para controle dinâmico:

repet(3){\n  aparec("Loop")\n}

Aula 7: Listas e Vetores

Manipula conjuntos de dados com push(), remove() e a contagem len():

cot lista = [1,2]\npush(lista, 3)

Aula 8: Funções (func)

Declare procedimentos com parâmetros e retornos antes/fora do bloco logical:

func soma(a, b){\n  return a +(b)\n}
